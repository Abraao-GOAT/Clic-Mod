CLIC MOD - MVP

Como abrir:
1. Extraia o arquivo ZIP.
2. Abra o arquivo index.html no navegador.
3. Use o botão Criar conta.
4. Vá em Publicar e cadastre uma roupa.

O que já funciona:
- Tela inicial
- Cadastro e login simples
- Feed de roupas
- Busca e filtro por Venda/Troca/Doação
- Publicar roupa com foto
- Perfil com pontos sustentáveis

Observação:
Este é um protótipo front-end. Os dados ficam salvos no navegador usando localStorage.
